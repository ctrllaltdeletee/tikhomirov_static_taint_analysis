# Артефакты ВКР: тестовые файлы, правила и скрипты для статического taint-анализа

Данный репозиторий содержит код, разработанный в рамках выпускной квалификационной работы на тему  
«Исследование границ детектирующей способности инструментов статического taint-анализа в контексте безопасности веб-приложений на Python».

**Автор:** Тихомиров Дмитрий Алексеевич, НИУ ВШЭ (МИЭМ), 2026.

## Структура репозитория

- `Gyp_*.py` — изолированные тестовые файлы для проверки гипотез.
- `custom_semgrep.yml` — пользовательские правила для Semgrep.
- `helpers.py` — вспомогательный файл с описанием пропагаторов для CodeQL.
- `metrics_script.py` — скрипт для расчёта метрик precision, recall, F1 по эталонной разметке.
- `scale_script.py` — скрипт для исследования масштабируемости CodeQL.
- `ground_truth.csv` — таблица эталонной разметки.

## Требования

- Python 3.10+
- Установленные анализаторы:
  - [Bandit](https://github.com/PyCQA/bandit)
  - [Semgrep](https://github.com/semgrep/semgrep)
  - [CodeQL CLI](https://github.com/github/codeql-cli-binaries/releases)

## Запуск

### 1. Расчёт метрик (Bandit, Semgrep, CodeQL)

Скрипт `metrics_script.py` автоматически запускает все три анализатора на тестовых файлах, вычисляет TP, FP, TN, FN, precision, recall, F1.

```bash
python3 metrics_script.py
```

### 2. Исследование масштабируемости CodeQL

Скрипт scale_script.py генерирует тестовые файлы, увеличивая их сложность и общий объем кодовой базы, измеряет время анализа и потребление памяти CodeQL, строит графики зависимостей.

```bash
python3 scale_script.py
```
